# 字库扩展至GB2312完整字符集

## 概述

本次更新将中文字库从 **1281个字符** 扩展至 **GB2312完整字符集（6766个字符）**，实现了对简体中文的完整支持。

## 更新详情

### 字符集扩展

| 项目 | 旧版本 | 新版本 |
|------|--------|--------|
| 字符数量 | 1,281个 | 6,766个 |
| 字库大小 | 40,151字节 (39.2KB) | 218,613字节 (213.5KB) |
| SPI Flash占用 | 1.90% | 10.4% |
| 字符集 | 自定义常用字 | GB2312完整字符集 |

### 字符集组成

- **一级汉字**: 3,755个（按拼音排序，常用字）
- **二级汉字**: 3,008个（按部首/笔画排序，次常用字）
- **自定义字符**: 3个（不在GB2312中的字符）
- **总计**: 6,766个汉字

### 空间利用

**旧中文信道名区回收**：
- 地址范围：0x020000 - 0x02FFFF（64KB）
- 状态：已废弃，数据已迁移至统一信道名区
- 用途：被新字库覆盖使用

**新字库布局**：
- 起始地址：0x010200
- 结束地址：0x045915
- 总大小：218,613字节（213.5KB）
- 剩余空间：1,769.7KB（SPI Flash总容量2MB）

## SPI Flash 地址分配

| 区域 | 起始地址 | 结束地址 | 大小 | 说明 |
|------|----------|----------|------|------|
| 字库位图数据 | 0x010200 | 0x038B18 | 162,384字节 | 6766个字符×12行×2字节 |
| 字库索引表 | 0x038B18 | 0x042A54 | 40,596字节 | 6766个条目×6字节 |
| 拼音索引表 | 0x042A54 | 0x0467D5 | 15,633字节 | 401个拼音音节 |
| 版本标记 | 0x045915 | 0x045915 | 1字节 | 版本号=3 |

## 使用说明

### 刷入字库

1. 使用Chrome/Edge打开刷机页面
2. 正常开机进入使用界面
3. USB连接电脑
4. 点击"刷字库"按钮

### 验证字库

字库刷入后，固件启动时会自动检测：
- 检查字库版本号（地址：0x042319）
- 版本匹配则跳过重复写入
- 版本不匹配则提示更新

## 技术参数

### 字库结构

```
地址范围          内容              大小
0x010200-0x038C07  位图数据         162,384字节
0x038C08-0x03F69F  Unicode索引表    27,064字节
0x03F6A0-0x043318  拼音表          15,633字节
0x043319           版本号          1字节
```

### 字符编码

- 编码方式：UTF-8
- 字符范围：U+4E00 - U+9FFF（基本汉字区）
- 点阵大小：12x12像素
- 存储格式：uint16_t数组（每行1个uint16_t）

## 性能影响

### 内存占用

- **Flash占用**: 增加160KB（从40KB到200KB）
- **RAM占用**: 无变化（运行时按需读取）
- **加载时间**: 无明显变化（索引查找O(log n)）

### 查找效率

- Unicode索引表：有序数组
- 查找算法：二分查找
- 时间复杂度：O(log n)
- 最坏情况：约13次比较（log₂6766 ≈ 12.7）
