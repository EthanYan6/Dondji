## 🧭 MAIN ONLY Mode

1. Press Menu
2. Go to Display (No.3)
3. Go to RxMode (No.2)
4. Select MAIN ONLY
5. Exit

---

## ⏱ Timer

Enable:

```bash id="en1"
-DENABLE_FEAT_F4HWN_RX_TX_TIMER=ON
```
