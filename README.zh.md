## ✨ 功能特性

* 基于 `F4HWN` 固件 `v5.3.1` 开发
* 仿 Motorola R7 UI（分支：`motorola_r7`）
* 双信道界面优化（参考 UV-KX-firmware）
* 经典手机风格图标菜单
* 锁屏界面重绘
* 菜单汉化
* 信道名称编辑逻辑调整，并支持仿手机九宫格（T9）输入

## 🧭 如何进入 MAIN ONLY

1. 按 Menu
2. 进入第 3 项：显示
3. 选择第 2 项：接收模式
4. 选择：仅主信道
5. 退出菜单

---

## ⏱ 计时功能

开启：

```bash id="zh1"
-DENABLE_FEAT_F4HWN_RX_TX_TIMER=ON
```
